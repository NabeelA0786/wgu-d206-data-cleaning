# D206 – Data Cleaning

**WGU Master of Science in Data Analytics**

## Overview
Assessed and cleaned a telecommunications customer churn dataset in preparation for downstream analysis. Addressed data quality issues including missing values, duplicates, and outliers.

## Dataset
Telecom customer churn dataset (~10,000 records, 50 features)

## What Was Done
- Identified and imputed missing values using mean, median, and mode strategies
- Detected and removed duplicate records
- Identified outliers using Z-score analysis
- Re-expressed categorical variables for analysis readiness
- Documented all cleaning decisions and produced a cleaned dataset

## Tools & Libraries
`Python` · `Pandas` · `NumPy` · `Jupyter Notebook`
